# Политика конфиденциальности (Privacy Policy)

Настоящая Политика объясняет, какие данные обрабатывает Discord-бот (далее — «Бот»), как они используются и какие права есть у Пользователя в соответствии с законодательством Российской Федерации (включая ФЗ-152 «О персональных данных»).

---

### 1. Цели обработки данных
Администрация обрабатывает данные исключительно для следующих целей:
* Обеспечение базового функционирования Бота (выполнение команд в реальном времени);
* Поддержка игровых и административных механик (хранение уровней, игрового прогресса, настроек серверов и системы предупреждений);
* Обеспечение безопасности (защита от спама и злоупотреблений);
* Сбор анонимной статистики для улучшения работы Бота.

### 2. Перечень собираемых данных
Бот может получать доступ и сохранять в базе данных следующую информацию:
* **Идентификаторы:** Discord User ID, Server (Guild) ID, Channel ID, Role ID;
* **Контент:** Текст сообщений (исключительно для работы систем логов модерации или специфических игровых команд) и метаданные (время и дата совершения действий);
* **Техническая информация:** Информация о ролях пользователя на сервере и его публичном профиле в Discord, необходимая для корректной работы прав доступа.

### 3. Срок хранения данных
Администрация придерживается принципа минимизации хранения данных:
* Данные хранятся до тех пор, пока это необходимо для обеспечения функционала.
* **Специальное условие:** После удаления Бота с Discord-сервера, все связанные с этим сервером данные хранятся в течение **7 (семи) календарных дней** (период ожидания на случай случайного удаления), после чего безвозвратно удаляются из базы данных.

### 4. Удаление данных и «Право на забвение»
Пользователь имеет полный контроль над своими данными. Предусмотрены следующие механизмы удаления:
* **Автоматизированный способ:** С помощью команды `/dmd` (Delete My Data) Пользователь может инициировать удаление данных. Команда позволяет выбрать: удалить данные только с конкретного сервера или полностью очистить информацию о пользователе во всей системе Бота.
* **Ручной способ:** Пользователь может направить запрос на удаление данных через электронную почту, указанную в разделе контактов.

### 5. Безопасность данных
Администрация принимает необходимые технические и организационные меры для защиты данных от несанкционированного доступа. Доступ к базе данных ограничен и предоставляется только для технической поддержки Бота. Данные не передаются третьим лицам, кроме случаев, прямо предусмотренных законодательством РФ.

### 6. Изменение политики
Администрация оставляет за собой право изменять настоящую Политику. Использование Бота после внесения изменений означает согласие с новой редакцией. Актуальный текст всегда доступен в данном репозитории.
